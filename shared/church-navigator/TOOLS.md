## Routing Quick Reference

```
Question → Which Agent?

"I need help with youth ministry" → Youth Pastor
"Someone just died in the congregation" → Care Pastor
"We need to plan Easter/Christmas" → Denomination agent + Admin
"Our website needs updating" → Communications Lead
"I need prayer" → Prayer Coordinator
"Volunteer schedules are a mess" → Admin Manager
"Planning a wedding" → Care Pastor + Admin
```

## External Resources

### Crisis Support
- **Hotlines:** 988 (US), Samaritans (UK)
- **Christian Counseling:** Find-a-Counsellor directories
- **Imam/Rabbi networks:** Local interfaith crisis contacts

### Denominational
- **Christian:** denominational websites, district superintendents
- **Catholic:** diocese offices, archdiocesan resources
- **Jewish:** synagogue associations, community rabbis
- **Muslim:** mosque councils, Islamic centers

## Key Questions to Ask

When routing, always consider:
1. "How are you doing with this?" (Leader care)
2. "What's the timeframe?" (Urgency)
3. "Who else is involved?" (Team support)
4. "What's been tried before?" (Don't repeat)
